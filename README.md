# cdump

`cdump` reads standard input and displays it with colours.
It always shows one byte with one coloured character.

## options

  - `-h` output html
  - `-l` show line number
  - `-r` respect newline
  - `-w`*width* wrap at *width* chars
  - `-W` do not show offset column
